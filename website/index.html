Setup Guide

1

Download & Extract

Download the SonaPrism.zip file and extract it anywhere on your computer.

2

Run Setup

Right-click setup.bat and choose "Run as administrator." It will download and install the required audio drivers automatically.

3

Install LoudMax

The setup script will open the LoudMax website. Download the VST2 64-bit version and put LoudMax64.dll in C:\Program Files\EqualizerAPO\VSTPlugins\

4

Restart & Launch

Restart your computer, then run SonaPrism from the desktop shortcut (right-click → Run as administrator).
